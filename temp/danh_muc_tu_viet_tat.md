# DANH MỤC TỪ VIẾT TẮT

| **Từ viết tắt** | **Nghĩa Tiếng Anh (Nếu có)** | **Nghĩa Tiếng Việt / Giải thích trong ngữ cảnh** |
| :--- | :--- | :--- |
| **AI** | Artificial Intelligence | Trí tuệ nhân tạo |
| **APT** | Advanced Persistent Threat | Mối đe dọa dai dẳng có chủ đích |
| **AUC-ROC** | Area Under the Receiver Operating Characteristic Curve | Diện tích dưới đường cong đặc trưng vận hành |
| **Bi-LSTM** | Bidirectional Long Short-Term Memory | Mạng nơ-ron bộ nhớ dài-ngắn hai chiều |
| **C&C** | Command and Control | Hệ thống điều khiển và ra lệnh (của mã độc) |
| **CDN** | Content Delivery Network | Mạng phân phối nội dung |
| **DGA** | Domain Generation Algorithm | Thuật toán sinh tên miền tự động |
| **DNS** | Domain Name System | Hệ thống phân giải tên miền |
| **DPI** | Deep Packet Inspection | Kiểm tra/Phân tích nội dung gói tin sâu |
| **EDR** | Endpoint Detection and Response | Hệ thống phát hiện và phản hồi tại máy trạm |
| **FN** | False Negative | Âm tính giả (Bỏ sót tấn công) |
| **FP** | False Positive | Dương tính giả (Báo động nhầm) |
| **FPR** | False Positive Rate | Tỷ lệ báo động giả |
| **GUI** | Graphical User Interface | Giao diện người dùng đồ họa |
| **HTTP(S)** | Hypertext Transfer Protocol (Secure) | Giao thức truyền tải siêu văn bản (bảo mật) |
| **IDS** | Intrusion Detection System | Hệ thống phát hiện xâm nhập |
| **IPS** | Intrusion Prevention System | Hệ thống ngăn chặn xâm nhập |
| **IP** | Internet Protocol | Giao thức Internet (Địa chỉ mạng) |
| **JSON** | JavaScript Object Notation | Định dạng dữ liệu văn bản |
| **LLM** | Large Language Model | Mô hình ngôn ngữ lớn |
| **MAC** | Media Access Control | Địa chỉ vật lý của thiết bị mạng |
| **ML** | Machine Learning | Học máy |
| **P2P** | Peer-to-Peer | Mạng lưới kết nối ngang hàng |
| **PCAP** | Packet Capture | Tập tin kết xuất lưu lượng gói tin mạng |
| **PPID** | Parent Process ID | Mã định danh tiến trình cha |
| **SIEM** | Security Information and Event Management | Hệ thống quản lý sự kiện và thông tin bảo mật |
| **SNI** | Server Name Indication | Thông tin định danh máy chủ (trong TLS) |
| **SOAR** | Security Orchestration, Automation and Response | Điều phối, tự động hóa và phản hồi an ninh |
| **SOC** | Security Operations Center | Trung tâm điều hành an ninh mạng |
| **TCP** | Transmission Control Protocol | Giao thức điều khiển truyền vận |
| **TLD** | Top-Level Domain | Tên miền cấp cao nhất (đuôi tên miền) |
| **TLS** | Transport Layer Security | Giao thức bảo mật tầng truyền tải |
| **TN** | True Negative | Âm tính thật (Luồng sạch đoán đúng) |
| **TP** | True Positive | Dương tính thật (Mã độc đoán đúng) |
| **TPR** | True Positive Rate | Tỷ lệ phát hiện đúng |
| **UDP** | User Datagram Protocol | Giao thức gói dữ liệu người dùng |
| **XGBoost** | eXtreme Gradient Boosting | Thuật toán học máy tăng cường Gradient tối ưu |
